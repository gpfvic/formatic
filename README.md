# 该项目用于公文格式自动生成


### flask.exe --app doc_format run
### flask --app doc_formate  run --debug
### pyinstaller打包成exe


## 集团公文格式要求
#### 标题(方正小标宋GBK二号，居中)
#### (空一行)
#### (2020年10月28日)(方正楷体GBK三号，居中)
#### (政策意见类加一行，标注“讨论稿”)
#### (空一行)
#### 正文一级标题(方正黑体GBK三号)
#### 正文二级标题(方正楷体 GBK三号)
#### 正文三级标题(方正仿宋GBK三号加粗)
#### 正文内容(方正仿宋GBK三号)
#### 其他规定:
#### 1、页面设置:上、下3厘米，左右 2.9 厘米;版式:页眉1.5厘米，页脚 1.75 厘米
#### 2、全文行距:固定值 29磅
#### 3、全文数字:TimesNewRoman三号
#### 4、正反面印刷，骑马订
#### 字号‘二号’对应磅值22,字号‘小二’对应磅值18,三号字体对应的数字是16磅
